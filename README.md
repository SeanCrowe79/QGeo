# QGeo
This project implements a numerical method due to Nielsen for finding geodesics on the special unitary group between the identity operator and a target unitary.

#Quick Start
To install this package clone the repo with:
git clone https://github.com/SeanCrowe79/QGeo.git
Navigate to the project directory and run
pip install -e .
This installs the module into your Conda environment. It is recommended to start with a fresh environment.
From there the example notebook can be used to generate geodesic control functions for your desired quantum circuit.
